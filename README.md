## 📝 Descripción

**Galactic Gadgets** es una tienda en línea temática de tecnología y gadgets extraterrestres. 

## 🔗 Enlaces

- **Repositorio GitHub**: [https://github.com/naty180482/ecommerce-frontend-m2](https://github.com/naty180482/ecommerce-frontend-m2)

## 📱 Páginas

- 🏠 **Home**: Grilla de productos con cards de Bootstrap
- 📦 **Detalle de producto**: Vista individual con imagen y descripción
- 🛒 **Carrito**: Listado de productos y resumen de compra

## 🛠️ Tecnologías

- HTML5 semántico (header, nav, main, section, article, footer)
- Bootstrap 5.3 (grid, componentes, utilidades)
- JavaScript ES6 (manipulación del DOM, eventos, localStorage)
- Git y GitHub (control de versiones)

## ✨ Funcionalidades

- ✅ Cards de productos generadas dinámicamente
- ✅ Botones "Agregar al carrito" con contador actualizado
- ✅ Persistencia del carrito con localStorage
- ✅ Diseño 100% responsivo (mobile-first)
- ✅ Navegación entre páginas
